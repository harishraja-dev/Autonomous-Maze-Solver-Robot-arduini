# Maze Solver Robot using Arduino

An autonomous maze-solving robot built using Arduino Uno, ultrasonic sensors, L298N motor driver, and DC motors.

The robot navigates through mazes using real-time wall detection and decision-making algorithms. The project combines robotics, embedded systems, fuzzy logic, and PID-based motion control for smooth and efficient navigation.

## Features

- Autonomous maze navigation
- Ultrasonic sensor based wall detection
- Fuzzy logic based turning control
- PID tuned straight-line movement
- Real-time obstacle detection
- Differential drive motor control

## Hardware Components

- Arduino Uno
- L298N Motor Driver
- Ultrasonic Sensors
- DC Motors
- Wheels & Chassis
- Rechargeable Battery
- LEDs
- Jumper Wires

## Software Used

- Arduino IDE
- Embedded C/C++

## Algorithms Used

- Wall Following
- Fuzzy Logic Control
- Fuzzy Tuned PID Control

## Working Principle

The robot continuously measures distances using ultrasonic sensors mounted on the left, right, and front sides.

- If no obstacle is detected ahead, the robot moves forward using PID-based correction.
- If a wall or obstacle is detected, fuzzy logic determines the turning direction and motor speed adjustment.
- The robot dynamically navigates the maze while maintaining stable movement.
